# DJ SNABU - Professional DJ Website

Huippuluokan DJ-sivusto, joka on nopea, tummanpuhuva ja integroituu some-alustoihin.

## Teknologiat

- **Astro** - Moderni web-framework
- **React** - Interaktiiviset komponentit
- **Tailwind CSS** - Utility-first CSS
- **TypeScript** - Type safety

## Kehitys

```bash
# Asenna riippuvuudet
npm install

# Käynnistä kehityspalvelin
npm run dev

# Rakenna tuotantoversio
npm run build

# Esikatsele tuotantoversio
npm run preview
```

- `MusicSection.astro` - SoundCloud-soitin
- `VideoGrid.astro` - YouTube-videot
- `InstagramGrid.astro` - Instagram-kuvat
